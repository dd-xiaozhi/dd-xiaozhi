# 🎨 GitHub Profile README 设置指南

## 🚀 快速设置(3步)

### 1️⃣ 创建特殊仓库
在 GitHub 创建名为 `dd-xiaozhi` 的仓库(与你的用户名相同)
- ✅ 设置为 Public
- ✅ 勾选 "Add a README file"

### 2️⃣ 上传文件
```bash
git clone https://github.com/dd-xiaozhi/dd-xiaozhi.git
cd dd-xiaozhi

# 复制 README 和 workflow 配置
cp /Users/xiaozhi/work/web/github_home_page/README.md .
cp -r /Users/xiaozhi/work/web/github_home_page/.github .

# 提交
git add .
git commit -m "✨ Add awesome profile README"
git push
```

### 3️⃣ 启用贪吃蛇动画
1. 打开仓库: https://github.com/dd-xiaozhi/dd-xiaozhi
2. 点击 **Actions** 标签
3. 点击 **I understand my workflows, go ahead and enable them**
4. 左侧选择 **Generate Snake** workflow
5. 点击右侧 **Run workflow** → **Run workflow**
6. 等待 1-2 分钟完成
7. 刷新你的主页: https://github.com/dd-xiaozhi
